Bonjour,

Votre assureur peut refuser votre sinistre, mais
dans beaucoup de cas ce refus est contestable.

Voici les étapes :
1. Vérifiez le motif exact dans la lettre de refus
2. Relisez votre contrat (conditions particulières)
3. Envoyez une réclamation LRAR au service contentieux
4. Demandez une expertise contradictoire si besoin
5. Saisissez le médiateur de l'assurance (gratuit)

Le délai de prescription est de 2 ans (article L114-1
du Code des assurances).
